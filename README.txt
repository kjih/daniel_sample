1) Install git. Create a github account. Configure ssh key. Clone the respoitory: https://github.com/kjih/daniel_sample
2) Download openjdk11
3) Download intellij community (IDE).
4) Download mysql. Create a new database. Create a new user and configure src/main/resources/application.properties
5) Compile project. Run "Application".
6) Install postman. Test the existing service: http://localhost:8080/person
7) Implement TODOs. Commit and push to github.
8) Create a "Notes" model with foreign key to the id of person table. Create a controller, service, etc. to handle create/read/update/delete of notes for persons.
